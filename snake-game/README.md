# スネークゲーム - Snake Game

クラシックなスネークゲームの現代的な実装。滑らかなアニメーション、複数の難易度、統計機能を備えています。

## 主な機能

### 🐍 ゲームプレイ
- **クラシックなスネークゲーム**
  - フードを食べて成長
  - 自分自身にぶつかるとゲームオーバー
  - スピードが徐々に上昇

- **2つのゲームモード**
  - クラシック：壁を抜けて反対側に出る
  - 壁あり：壁に当たるとゲームオーバー

- **3つの難易度**
  - 簡単：ゆっくりスタート
  - 普通：バランスの良い速度
  - 難しい：高速スタート

### 🎮 操作方法
- **矢印キー** - スネークの方向転換
- **SPACE** - 一時停止
- **マウス** - ボタン操作

### 📊 得点システム
- フード1個：10点
- 長さが増えるほど速度上昇
- 5匹成長ごとにスピードレベルアップ

### 🏆 統計機能
- ハイスコア記録
- 総ゲーム数
- 最高長さ
- 獲得フード総数
- LocalStorageで永続化

### 🎯 達成システム
- 長さ10、25、50、100到達
- スピードレベルアップ通知
- 新記録達成通知
- リアルタイム表示

## ゲームルール

### 基本ルール
1. 矢印キーでスネークを操作
2. 赤いフードを食べると成長
3. 自分自身にぶつかるとゲームオーバー
4. 長くなるほど難しくなる

### クラシックモード
- 壁にぶつかると反対側から出現
- より長く生き延びることが可能

### 壁ありモード
- 壁にぶつかるとゲームオーバー
- より慎重なプレイが必要


## 攻略のコツ

1. **中央を使う** - 端に追い詰められないように
2. **渦巻き移動** - 効率的に空間を使う
3. **急がない** - 焦ると自爆しやすい
4. **先読み** - 次の動きを常に考える
5. **パターン作り** - 安定した移動パターンを作る
