結果

あみだくじの遊び方

あみだくじとは

あみだくじは、複数の縦線と横線で構成された図を使って、公平に物事を決める日本の伝統的なくじ引き方法です。

ルール

  1. 上部の選択肢から1つを選びます
  2. 選んだ縦線を上から下へ辿ります
  3. 横線に当たったら、必ずその横線に沿って隣の縦線に移動します
  4. 下まで辿り着いた先の景品があなたの結果です

使い方

  • 縦線・横線の数: 参加人数や景品の数に合わせて設定できます
  • 選択肢設定: 上部に表示される選択肢を自由に編集できます
  • 景品設定: 下部に表示される景品を自由に編集できます
  • 自動生成: 「新しいくじを作成」で自動的にランダムな配置が生成されます
