# Productivity Pro - タスク管理 & ポモドーロアプリ

プロフェッショナルな生産性管理アプリケーション。タスク管理とポモドーロテクニックを統合した高機能Webアプリです。

## 主な機能

### タスク管理
- 優先度別タスク管理（高・中・低）
- タスクのフィルタリング（すべて・進行中・完了）
- タスクの完了/未完了切り替え
- タスク作成時刻の表示
- ドラッグ不要のシンプルなUI

### ポモドーロタイマー
- カスタマイズ可能な作業時間と休憩時間
- ビジュアルプログレスサークル
- サウンド通知
- 自動休憩開始オプション
- 今日/合計のポモドーロカウント

### 統計情報
- タスク完了率の可視化
- アクティビティログ
- リアルタイム統計更新
- 美しいグラフィカル表示

### その他
- ダークモード/ライトモード切り替え
- LocalStorageによるデータ永続化
- レスポンシブデザイン
- モダンなアニメーション効果
- 直感的なナビゲーション

## 使用技術

- **HTML5** - セマンティックマークアップ
- **CSS3** - CSS Variables、Grid、Flexbox、アニメーション
- **Vanilla JavaScript** - ES6+クラス構文、LocalStorage API
- **SVG** - アイコンとグラフィック
- **Web Audio API** - タイマー通知音

## アーキテクチャ

- **DataManager クラス** - データの永続化と管理
- **PomodoroTimer クラス** - タイマー機能の実装
- コンポーネントベースの設計
- イベント駆動型アーキテクチャ

## 起動方法

1. ブラウザで `index.html` を開く
2. サーバー不要で即座に動作

## カスタマイズ

CSS変数を編集することで、簡単にカラースキームをカスタマイズできます：

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* その他の変数... */
}
```
