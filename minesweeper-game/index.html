マインスイーパーの遊び方

ゲームの目的

地雷のないマスをすべて開けることが目標です。地雷を踏むとゲームオーバーです。

操作方法

  • 左クリック - マスを開く
  • 右クリック - 旗を立てる（地雷マーク）
  • 右クリック×2 - はてなマーク（？）
  • 右クリック×3 - マークを解除

数字の意味

開いたマスに表示される数字は、そのマスの周囲8マスにある地雷の数を示します。

攻略法

  1. 数字を見て、確実に地雷がある場所に旗を立てる
  2. 数字を見て、確実に安全な場所を開く
  3. パターンを覚えて効率よく開く
  4. 角や端から攻めると安全

難易度

  • 初級 - 9×9マス、地雷10個
  • 中級 - 16×16マス、地雷40個
  • 上級 - 16×30マス、地雷99個
