# FinanceHub - プロフェッショナル金融ダッシュボード

エンタープライズグレードの金融管理ダッシュボードアプリケーション。リアルタイムチャート、ポートフォリオ管理、詳細分析機能を備えた本格的なWebアプリです。

## 主な機能

### 📊 ダッシュボード
- **リアルタイム統計**
  - 総資産、投資利益、月間支出、貯蓄率の可視化
  - 前月比較と増減率表示
  - アニメーション付きカード

- **インタラクティブチャート**
  - 資産推移の折れ線グラフ
  - 資産配分のドーナツチャート
  - カスタム期間選択（1ヶ月/3ヶ月/6ヶ月/1年）

- **最近の取引**
  - 購入/売却/配当の履歴表示
  - アイコンとカラーコーディング
  - 日付と金額の詳細表示

- **マーケット情報**
  - 主要指数のリアルタイム表示
  - 日経平均、S&P 500、ビットコイン、為替
  - 増減率とトレンド表示

### 💼 ポートフォリオ管理
- 資産タイプ別管理（株式、暗号資産、債券、投資信託）
- 損益計算と利益率表示
- 現在価値の自動計算
- 資産追加モーダル
- カラーコーディングされたカード

### 📈 取引履歴
- 完全な取引履歴テーブル
- フィルタリング機能（購入/売却/配当）
- 日付フィルター
- ステータス表示（完了/保留中）
- ソート可能なカラム

### 📉 詳細分析
- **月間収支チャート** - 棒グラフで収支推移を表示
- **カテゴリ別支出** - ドーナツチャートで支出内訳を可視化
- **投資パフォーマンス** - 折れ線グラフで運用成績を追跡
- **リスク分析** - シャープレシオ、ボラティリティ、ドローダウン、ベータ値

## 技術的特徴

### アーキテクチャ
- **フロントエンド**: Vanilla JavaScript (ES6+)
- **データ管理**: LocalStorage による永続化
- **チャート**: カスタムCanvasベースのチャートライブラリ
- **スタイリング**: CSS Variables による完全なテーマ対応

### カスタムチャートライブラリ
`chart.js`に独自の軽量チャートライブラリを実装：
- 折れ線グラフ（グラデーション、アニメーション）
- ドーナツチャート（対話的凡例）
- 棒グラフ（カスタムカラー）
- レスポンシブ対応（デバイスピクセル比考慮）

### データ構造
```javascript
{
  portfolio: [
    {
      id: number,
      type: 'stock' | 'crypto' | 'bond' | 'fund',
      name: string,
      symbol: string,
      quantity: number,
      price: number,
      currentPrice: number
    }
  ],
  transactions: [
    {
      id: number,
      date: ISO8601,
      type: 'buy' | 'sell' | 'dividend',
      symbol: string,
      name: string,
      quantity: number,
      price: number,
      status: 'completed' | 'pending'
    }
  ]
}
```

### デザインシステム
- **カラースキーム**: CSS Variablesによる一元管理
- **ダークモード**: 完全対応（自動保存）
- **レスポンシブ**: モバイル、タブレット、デスクトップ
- **アニメーション**: スムーズなトランジション
- **アクセシビリティ**: セマンティックHTML、適切なコントラスト

## 使用方法

### 起動
1. ブラウザで`index.html`を開く
2. サーバー不要で即座に動作

### 資産追加
1. 「ポートフォリオ」タブに移動
2. 「資産を追加」ボタンをクリック
3. 資産タイプ、銘柄名、数量、価格を入力
4. 「追加」をクリック

### データ管理
- すべてのデータはLocalStorageに自動保存
- ブラウザを閉じても永続化
- デフォルトデータで初期化可能

## カスタマイズ

### テーマカラー変更
`style.css`の`:root`セクションでカラーを変更：

```css
:root {
    --primary: #667eea;
    --secondary: #764ba2;
    --success: #10b981;
    --danger: #ef4444;
    /* ... */
}
```

### チャートデータ
`app.js`の各`render*`関数内でデータをカスタマイズ
