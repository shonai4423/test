# 🎮 ゲーム＆アプリポータル v1.0.0

## 📅 リリース日
2025年1月

## 🎉 初回リリース

ゲーム＆アプリポータルの初回リリースです。様々なゲームとアプリケーションを1つのポータルサイトからアクセスできます。

## ✨ 含まれるコンテンツ

### 📱 アプリケーション
- **天気予報アプリ** - 世界中の都市の天気情報をリアルタイムで確認
- **生産性アプリ** - タスク管理とポモドーロタイマー
- **財務ダッシュボード** - 資産管理と取引履歴の可視化

### 🎯 クラシックゲーム
- **スネークゲーム** - クラシックなスネークゲーム
- **マインスイーパー** - 3つの難易度から選択可能
- **ブロック崩し** - パワーアップ機能付き
- **テトリス** - ホールド機能搭載
- **パックマン** - 4体のゴーストから逃げろ

### 🎲 ボードゲーム
- **オセロ** - AIと対戦、3つの難易度
- **花札** - 日本の伝統カードゲーム、11種類の役

## 🎨 特徴
- モダンで美しいUIデザイン
- ダークモード/ライトモード対応
- レスポンシブデザイン（スマホ対応）
- すべてのゲームとアプリがブラウザで動作
